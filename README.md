# ZI Auto Clicker

ZI Auto Clicker is an advanced Windows auto-clicker and key/mouse automation tool. with PyQt5 GUI and allows you to automatically click or hold keyboard and mouse buttons. Customize CPS (Clicks per Second), and timers for precise automation. Works in selected application windows or globally.

---

## Features

- Low-level hooks (can use same buttons)
- Select Trigger Key
- Select Affected Key
- Auto-click or Hold Down modes
- Hold / Toggle start options
- CPS (Clicks per Second) adjustment
- Limit functionality to a specific application window
- Timer with configurable hold/release durations
- Easy-to-use GUI
- Disable key
